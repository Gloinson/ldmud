LDMud

Overview

LDMud is a virtual machine, telnet server and compiler for the LPC language. It is a direct descendent of the original LPMud gamedriver.

License

LDMud is licensed as open source but may not be used for monetary gain. For details see COPYRIGHT.

Similar projects

  • FluffOS is a fork of MudOS which is a fork of LPMud 3.1.2.
  • Dworkin's Game Driver is a reimplementation from scratch of LPMud released under an open source license allowing commercial use.

Note that each driver implements its own LPC dialect that may not be compatible with the others.
